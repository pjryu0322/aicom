# T05 회의록 작성·관리 자동화 (프로토타입)

녹취 파일 업로드 → 텍스트 변환 → 화자 분리 → 회의록 초안 생성 → 수정 요청 → 최종 승인 → 공유/배포 흐름을 **정적 목업**으로 구현한 React + Tailwind 프로토타입입니다.

## 실행 방법

```bash
cd web
npm install
npm run dev
```

브라우저에서 안내된 로컬 주소로 접속합니다.

## 주요 페이지

- `자료 업로드`: `/upload`
- `변환·처리 상태`: `/processing`
- `회의 분석 워크스페이스(3컬럼)`: `/workspace`
- `문서 편집`: `/editor`
- `승인 요청·함`: `/approval`
- `공유·배포`: `/share`

## 워크스페이스(요구사항 반영)

- 좌측 사이드바: 회의 파일 목록 / 참여자·화자 목록 / 작업 상태
- 중앙: 대화·작업 타임라인 / 업로드 카드(목업) / AI 변환 상태(진행률) / 하단 메시지 입력(목업)
- 우측 패널: `요약본`/`스크립트` 탭 전환
  - 요약본: 핵심 안건 / 결정사항 / 할 일
  - 스크립트: 화자별 발언 목록

## 참고

- 백엔드 없음(업로드/저장/공유 링크 등은 모두 목업 UI).
- 목업 데이터는 `src/lib/mockData.ts`에 있습니다.
